# EarthNaruMac

A tiny macOS floating companion that shows a pixel Earth character and the current time.

The app no longer watches keyboard input. The character changes its mood and motion over time:

- Dawn: sleepy wake-up, stretching, sunrise motion.
- Morning: energetic waving, hopping, and star play.
- Noon: brighter, busier motions.
- Afternoon: calmer looking-around and workday pacing.
- Evening: relaxed waves and sparkle motions.
- Night: slower breathing, sleepy eyes, and moonlit drifting.
- Late night: mostly sleeping and dreaming.

## How to run in Xcode

1. Open `macos/EarthNaruMac.xcodeproj`.
2. Select the `EarthNaruMac` scheme.
3. Run the app.

No Accessibility or Input Monitoring permissions are required.
